# Results Overview

This directory contains representative outputs from the SVS13A analysis pipeline.

## Gaussian_fitting

Spectral fitting products, including:

- `1D/`: single-transition Gaussian fitting results
- `2D/`: two-dimensional fitting results
- `joint_fitting_1G/`: joint fitting across multiple transitions with one Gaussian component
- `joint_fitting_2G/`: joint fitting across multiple transitions with two Gaussian components
- `multi_G/`: multi-component Gaussian fitting results
- `select_target/`: target-selection figures and tables

These subdirectories include:
- `fit_pages_*.pdf`: visual summaries of fitting results
- `fit_results_*.csv`: fitted parameter tables

## rotational_diagram

Rotation-diagram temperature estimates and temperature maps.

## rd_dual

Two-line ratio temperature estimates.

## pv_diagrams

Representative position–velocity diagram products.

## Note

Large MCMC corner-plot outputs are not included in this repository in order to keep the GitHub repository lightweight and easier to browse.
